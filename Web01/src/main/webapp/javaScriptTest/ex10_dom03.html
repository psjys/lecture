** DOM3_JS 단계별 실행절차 **

1. 파싱단계 : 구문분석 단계
	=> 선언적 함수들과, 명시적(var, let, const) 전역변수들을 정의 
	=> 함수먼저, 다시 변수 정의 
	=> 선언적 함수들은 값(body)도 할당함 
	=> 변수(객체형, 익명함수형 포함)들은 이름만 정의하고 값은 실행시에 할당 
	=> 그러므로 파싱 단계에서 변수는 값이 없고, 익명함수와 객체는 undefined 값을 가짐. 

2. 실행단계
	=> 순차적으로 실행 (함수의 정의는 건너뜀)

3. 변수와 메모리
	=> 변수명 : stack 영역 
	=> 변수의값 : heap 영역
